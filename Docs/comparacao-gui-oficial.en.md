## Summary
Cockpit is a clean webview client that drives the Claude Code CLI. It does the **conversation core** well (streaming, thinking, tool cards with per-tool rendering, markdown, highlight), **model/effort with live discovery**, **stats panel** (context/cost/cache/tokens), **history/resume/delete session**, **image+file paste**, **permission modal**, **pt-BR/en i18n**, **synced theme**.

**Biggest gap:** **side-by-side inline diff** review (today we only approve edits via a permission modal with raw JSON). Also missing: slash menu, @-mentions/selection, reviewable plan mode, checkpoints/rewind, MCP, plugins, subagents, hooks, skills, output-styles, statusline, onboarding, browser, Jupyter, remote control, multi-tab, sign-in.

**Two items built-but-not-wired:** 5h/7d limit bars show "Not reported"; context breakdown is a stub.

## Our extras (only Cockpit has)
- **Bilingual pt-BR/en i18n** (host + webview) with runtime switching, no reload
- **Scroll-markers rail** (minimap): 1 marker per user prompt, with numbered hover
- **Local cost estimate** from the model's pricing table, labeled "estimated" when no real cost exists
- **Dedicated cache panel**: hit-rate, read, write
- Always-visible input/output counters + collapsible stats panel
- **Live model discovery** via `/v1/models` + recording the active model from the init event
- **Grouped** model combo (Aliases / Versions / 1M) + Custom entry + friendly names
- **Auto-resume** of the folder's last session on open
- **File paste as path** + authoritative Unicode read on Windows (FileDropList)
- Per-tool emoji icons + rich render per type (Bash split, Read gutter, Write/Edit highlight)
- Line-number gutter in Read, next to the clean highlighted code
- **Rich tooltips** with a colored icon badge (title + description)
- Consolidated status bar (idle/busy/error dot + active model chip)
- **Elegant confirmation dialog** in the webview to delete a session (danger, Esc/overlay)
- CLI presence/version detection + disabled composer + cmd-safe spawn on Windows
- **Custom path** setting for the `claude` binary
- **Tolerant stream-json parser** across CLI version changes
- High-context alert (85%)
- Manual context limit override (auto 1M/200K derived from the active model)
